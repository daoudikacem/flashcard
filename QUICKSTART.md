# دليل البدء السريع - Quick Start Guide

## كيفية تشغيل التطبيق

### الخطوة 1: فتح المشروع
1. افتح Android Studio
2. اختر "Open an Existing Project"
3. اختر مجلد المشروع: `e:\تعليم\PROGRAMMATION\flashcard`
4. انتظر حتى ينتهي Gradle Sync

### الخطوة 2: تشغيل التطبيق
1. وصّل جهاز Android أو شغّل المحاكي
2. اضغط على زر Run (▶) أو `Shift + F10`
3. اختر الجهاز
4. انتظر حتى يتم تثبيت التطبيق

### الخطوة 3: استخدام التطبيق

#### إنشاء أول مجموعة
1. افتح التطبيق
2. اضغط على زر **+** في أسفل الشاشة
3. أدخل اسم المجموعة (مثال: "الإنجليزية")
4. أدخل وصف (اختياري)
5. اضغط "إنشاء"

#### إضافة بطاقات
1. اضغط على المجموعة التي أنشأتها
2. اضغط على زر **+**
3. أدخل السؤال (مثال: "ما معنى Book؟")
4. أدخل الجواب (مثال: "كتاب")
5. فعّل "إنشاء بطاقة معكوسة" إذا أردت
6. اضغط "إنشاء"
7. كرر العملية لإضافة المزيد

#### الدراسة
1. من الشاشة الرئيسية، اضغط على أيقونة **▶** بجانب المجموعة
2. اختر عدد البطاقات (3، 5، 10، أو 20)
3. اضغط "بدء الدراسة"
4. اقرأ السؤال على البطاقة
5. اضغط على البطاقة لقلبها وإظهار الجواب
6. قيّم الصعوبة:
   - **سهل**: تعرفت على الجواب بسهولة
   - **متوسط**: احتجت للتفكير قليلاً
   - **صعب**: لم أكن أعرف الجواب
7. اضغط "التالي" أو قيّم البطاقة للانتقال للتالية
8. بعد الانتهاء، اضغط "الدفعة التالية" للمزيد

#### الامتحان
1. اضغط على أيقونة **🎓** في أعلى الشاشة
2. اختر مجموعة أو أكثر
3. اضغط "بدء الامتحان"
4. اقرأ السؤال
5. اضغط "إظهار الجواب"
6. قيّم إجابتك:
   - **صحيح**: عرفت الجواب
   - **خطأ**: لم أعرف الجواب
7. أكمل جميع الأسئلة
8. شاهد نتيجتك النهائية

#### الإحصائيات
1. اضغط على أيقونة **📊** في أعلى الشاشة
2. شاهد:
   - عدد المجموعات
   - عدد البطاقات
   - توزيع الصعوبة
   - معدل النجاح
   - إجمالي المراجعات

## نصائح للاستخدام الفعال

### 1. ابدأ صغيراً
- أنشئ مجموعة واحدة أولاً
- أضف 5-10 بطاقات
- اختبر وضع الدراسة

### 2. استخدم البطاقات المعكوسة
- فعّل الخيار عند إنشاء البطاقة
- ستحصل على بطاقتين بدلاً من واحدة
- مفيد لتعلم اللغات

### 3. راجع يومياً
- استخدم وضع الدراسة يومياً
- الخوارزمية ستتعلم من تقييماتك
- الأسئلة الصعبة ستظهر أكثر

### 4. قيّم بصدق
- لا تخف من اختيار "صعب"
- هذا يساعد الخوارزمية على مساعدتك
- التقييم الصادق يحسّن النتائج

### 5. استخدم الامتحان للاختبار
- امتحن نفسك بعد الدراسة
- حاول تحقيق 70% أو أكثر
- أعد الامتحان لتحسين نتيجتك

## حل المشاكل الشائعة

### المشكلة: التطبيق لا يعمل
**الحل**:
1. تأكد من تحميل جميع التبعيات
2. نظّف المشروع: `Build > Clean Project`
3. أعد البناء: `Build > Rebuild Project`
4. شغّل مرة أخرى

### المشكلة: البطاقات لا تظهر
**الحل**:
1. تأكد من إضافة بطاقات للمجموعة
2. أعد تحميل الشاشة
3. تحقق من قاعدة البيانات

### المشكلة: الخوارزمية لا تعمل بشكل صحيح
**الحل**:
1. تأكد من تقييم البطاقات بعد كل مراجعة
2. انتظر حتى تتراكم البيانات
3. الخوارزمية تتحسن مع الوقت

## التعديل والتطوير

### تغيير الألوان
افتح `MainActivity.kt` وعدّل `colorScheme`:
```kotlin
lightColorScheme(
    primary = Color(0xFF6200EE),    // اللون الأساسي
    secondary = Color(0xFF03DAC5),  // اللون الثانوي
    tertiary = Color(0xFF6200EE)    // اللون الثالث
)
```

### تعديل الخوارزمية
افتح `SpacedRepetitionAlgorithm.kt` وعدّل:
```kotlin
private val INITIAL_INTERVALS = mapOf(
    Difficulty.EASY to 3 * DAY,     // تغيير فترة السهل
    Difficulty.MEDIUM to 1 * DAY,   // تغيير فترة المتوسط
    Difficulty.HARD to 10 * MINUTE  // تغيير فترة الصعب
)
```

### إضافة شاشة جديدة
1. أنشئ ملف في `ui/screen/`
2. أضف route في `navigation/Screen.kt`
3. أضف composable في `navigation/AppNavigation.kt`

## الدعم

2. اقرأ التعليقات في الكود
3. جرّب التعديلات البسيطة أولاً
