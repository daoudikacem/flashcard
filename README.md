# تطبيق البطاقات التعليمية - FlashCard App

تطبيق Android حديث للحفظ الذكي باستخدام نظام التكرار المتباعد (Spaced Repetition)

## المميزات

✅ نظام المجموعات المتعددة  
✅ بطاقات تعليمية مع خيار البطاقة المعكوسة  
✅ وضع الدراسة مع خوارزمية ذكية  
✅ وضع الامتحان مع تقييم الأداء  
✅ إحصائيات تفصيلية  
✅ تصميم Material Design 3  
✅ دعم الوضع الليلي  
✅ دعم كامل للعربية RTL  
✅ استيراد وتصدير البيانات  

## التثبيت

1. افتح المشروع في Android Studio
2. انتظر تحميل جميع التبعيات
3. شغل التطبيق على جهاز أو محاكي

## البناء

```bash
./gradlew assembleDebug
```

## الهيكل

```
data/          → قاعدة البيانات والمستودعات
domain/        → نماذج الأعمال والخوارزميات
ui/            → الشاشات ونماذج العرض
navigation/    → نظام التنقل
di/            → حقن التبعية
```

## الخوارزمية

التطبيق يستخدم خوارزمية Spaced Repetition مبسطة مشابهة لـ Anki:

- **صعب**: مراجعة كل 10 دقائق (معامل 1.2x)
- **متوسط**: مراجعة كل يوم (معامل 1.5x)
- **سهل**: مراجعة كل 3 أيام (معامل 2.5x)

## التقنيات

- MVVM Architecture
- Coroutines & Flow
